Programming Assignment 5

Game Library/Review/Showcase

Description

This is an implementation of an idea that I have had for a while now. This project serves as a way I might be able to post personal reviews of games I have played.

This project is a "library" of various games that I have played. The user can scroll through the library to explore these games.

When scrolling through the library, game cases modeled after the typical case that video game CDs come in will fly past the user.

If the user wants to take a closer look at one of the games, they just have to click on the game case object, and that will bring that specific case closer to the camera.

When inspecting the cases, a text box would appear to the right that contained my personal thoughts/reviews/ratings/jokes of the game in question.

Implementation

The most important library used in this project is Three.js. This library handles all of the heavy lifting for the graphics and rendering.

Compared to working directly with WebGL, using Three.js is much much easier, since all of the rendering/modeling/projection/viewing code has already been written for you. All you have to do is to build your scene.

The second most important library used is tween.js. This is a library that provides animateable interpolation curves that can be used to animate all kinds of things.

I used tween.js to handle the rotations and translations of the various models on the screen. In particular, I made liberal use of the Exponential curve when handling the position animations of the objects, since that curve provided the kind of motion I was really going for when envisioning this project.

Each game case model is made of a simple cube with adjusted proportions. The textures were pulled off of the internet and pasted onto the proper sides of the cube to convey the appearance of a game case.

Instructions/Tips

To scroll through the "library" of games, simply use the scroll wheel on the mouse or an equivalent input. This will allow you to look through the library with cool animations for each game case as you scroll.

To select a game case for closer viewing, simply click on the desired game. This will bring the game case close to the camera for better viewing.

While inspecting each game case, a short blurb containing my thoughts about the game will appear to the right of the game case.
